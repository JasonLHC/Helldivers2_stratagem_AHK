# Helldivers 2 Stratagem AHK Script

AutoHotkey script for fast and convenient stratagem input automation in Helldivers 2.

## Features

- Numpad hotkeys for quick stratagem execution
- Simple and lightweight
- Easy to modify key sequences
- Designed for AutoHotkey v2

## Requirements

- Windows
- AutoHotkey v2 installed

Download AutoHotkey:
https://www.autohotkey.com/

## Installation

1. Clone or download this repository.
2. Install AutoHotkey v2.
3. Double-click `helldiver.ahk` to run.

## Usage

- Run the script.
- Press configured hotkeys (e.g., Numpad keys).
- Stratagem sequences will be entered automatically.

## Configuration

Open the `.ahk` file in a text editor to customize key bindings or sequences.
